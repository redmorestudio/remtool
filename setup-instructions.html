RemTool Library Setup

To complete the RemTool setup, you need to download the following libraries and place them in the lib/ folder:

Required Libraries:

  1. PDF.js (for reading and displaying PDFs)
    • Download from: pdf.js
    • Download from: pdf.worker.js
    • Save as: lib/pdf.js and lib/pdf.worker.js
  2. PDF-lib (for creating and modifying PDFs)
    • Download from: pdf-lib.min.js
    • Save as: lib/pdf-lib.min.js

Alternative: Use CDN Links

Alternatively, you can modify index.html to use CDN links instead of local files:

    <!-- Replace the local script tags with: -->
    <script src="https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.min.js"></script>
    <script>
        pdfjsLib.GlobalWorkerOptions.workerSrc = 'https://cdnjs.cloudflare.com/ajax/libs/pdf.js/3.11.174/pdf.worker.min.js';
    </script>
    <script src="https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js"></script>

To Run RemTool:

  1. Open a terminal in the remtool directory
  2. Start a local web server:
    • Python 3: python3 -m http.server 8000
    • Python 2: python -m SimpleHTTPServer 8000
    • Node.js: npx http-server -p 8000
  3. Open your browser to: http://localhost:8000

API Keys Required:

You'll need at least one of these API keys:

  • Grok API Key (required): Get from X.AI
  • Gemini API Key (optional): Get from Google AI Studio
  • Claude Sonnet API Key (optional): Get from Anthropic
